Pytania dotyczące wewnętrznego arkusza stylów

  1. Gdzie jest wstawiany?

    2. Wewnętrzny arkusz stylów jest wstawiany bezpośrednio w sekcji <head> dokumentu HTML.

  2. Składnia teoretyczna
    3. <style>
    [selektor] {
    [cecha]: [wartość];
    [cecha]: [wartość];
          }
    </style>
  3. Jakie mogą być selektory?

    4. Selektory mogą być tagami HTML, klasami, identyfikatorami, a także kombinacjami tych elementów.

  4. Jakie mogą być cechy i wartości?

    5. Cechy mogą być różnymi właściwościami CSS, takimi jak color, font-size, margin, border itp. Wartości mogą być różnymi wartościami dla tych właściwości, na przykład konkretne kolory, rozmiary czcionek, odstępy, itp.
